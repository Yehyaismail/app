## Problem Statement
موقع تواصل محادثات شبيه بالواتساب

## Architecture
- Backend: FastAPI + MongoDB + Emergent Object Storage
- Frontend: React + Tailwind + Shadcn UI
- Auth: JWT httpOnly cookies with refresh tokens
